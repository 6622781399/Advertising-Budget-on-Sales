# Advertising Budget on Sales

## Project Overview

This project analyzes the relationship between advertising budgets (TV, Radio, and Newspaper) and product sales. The objective is to explore the data, identify patterns, and build predictive models to estimate sales based on advertising spending.

## Dataset
The dataset contains the following variables:
* **TV**: Advertising budget spent on TV
* **Radio**: Advertising budget spent on Radio
* **Newspaper**: Advertising budget spent on Newspaper
* **Sales**: Product sales

## Project Steps
### 1. Exploratory Data Analysis (EDA)
* Data cleaning and preprocessing
* Checking missing values
* Data visualization (scatter plots, correlations)
* Identifying relationships between features and sales

### 2. Model Development
* Linear Regression model
* Feature selection
* Model training and evaluation

### 3. Model Evaluation
* Performance metrics:
  * Mean Absolute Error (MAE)
  * Mean Squared Error (MSE)
  * R-squared (R²)

## Results
* TV and Radio advertising budgets show strong positive correlation with sales
* Newspaper has weak or negligible impact
* The regression model provides a good fit for predicting sales
